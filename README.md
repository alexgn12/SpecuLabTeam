# SpecuLabTeam

Repositorio del equipo SpecuLab para el Summer Camp 2025.

Este proyecto incluye una API desarrollada en .NET 8, junto con módulos para servicios, lógica de negocio y acceso a datos. El objetivo es facilitar la gestión y experimentación de solicitudes relacionadas con edificios y apartamentos, integrando autenticación, autorización y comunicación en tiempo real.

Incluye:
- API RESTful con autenticación JWT.
- Integración con servicios externos.
- Gestión de usuarios y roles.
- Comunicación en tiempo real con SignalR.
- Documentación y pruebas con Swagger.
- Intelijencia artificial para mejorar la experiencia del usuario.
